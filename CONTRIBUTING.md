# Contributing to Email Signature Online

Thank you for your interest in contributing. This project aims to provide a premium, easy-to-use email signature generator for everyone.

## Development Workflow

1. Fork the repository.
2. Clone your fork: `git clone https://github.com/your-username/email-signature-online.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feat/your-feature-name`
5. Start the development server: `npm run dev`
6. Make your changes and ensure they follow the project's aesthetic and code quality standards.
7. Commit your changes: `git commit -m "feat: add your feature description"`
8. Push to your fork: `git push origin feat/your-feature-name`
9. Open a Pull Request.

## Code Standards

- **Aesthetics**: Maintain the premium, minimalist design system.
- **Performance**: Ensure a buttery-smooth 60fps experience. Avoid heavy external dependencies.
- **Accessibility**: Use semantic HTML and ensure high contrast for visibility.
- **Localization**: Add or update translations in `src/dictionaries` if adding new text.

## Reporting Issues

Please use GitHub Issues to report bugs or suggest new features. Provide as much context as possible, including steps to reproduce bugs.
